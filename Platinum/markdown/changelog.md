-   修复了键盘导航顺序错误的问题。
-   优化了悬浮提示的显示效果。

-   **新增流畅模式，专为低性能电脑打造，使用不卡顿。**

#### 更新

-   **新增 Platinum Insider，加入的人能够收到预览版更新。**
-   **修复了更新模块的一些逻辑错误。**
-   更新下载完毕后显示重启对话框，提醒用户进行重启。

#### 网页

-   查看证书现已支持显示证书链。
-   修复了在暗色模式下，部分网页背景错误显示为白色的问题 (例如 Twitter 等)。

#### 设置

-   一些样式上的修改。

#### 辅助功能

-   **浏览器侧边栏 UI (例如设置、历史记录等) 现已兼容屏幕阅读器！**
